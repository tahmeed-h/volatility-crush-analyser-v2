# Volatility Crush Analyzer

A Python application that analyses option volatility crush scenarios using real market data.

## Features
- Black-Scholes option pricing implementation
- Calculation of Greeks (Delta, Gamma, Vega, Theta)
- Scenario analysis for volatility changes
- Straddle pricing and P/L simulation
- Interactive Brokers API integration
- GUI built using Tkinter

## Technologies
Python, NumPy, Pandas, SciPy, Interactive Brokers API

## Use Case
Helps traders analyse potential profit or loss from volatility crush strategies around earnings events.
